# Polytechnic Course Registration System (PCRS)

## 📌 Overview

This project is a PHP-based web application developed to manage course registration for students and administrators.

## 🚀 Features

* Student registration & login
* Admin login
* Course browsing and registration
* Role-based dashboards
* Secure authentication system

## 🛠 Technologies Used

* PHP (Core PHP)
* MySQL
* HTML, CSS, JavaScript (AJAX)
* Bootstrap

## ⚙️ Setup Instructions

1. Upload files to a web server (e.g., Hostinger)
2. Create a MySQL database
3. Import the provided `.sql` file
4. Configure `config.php` with your database credentials

## 🔐 Default Access

* Admin: (manually inserted in database)
* Student: Register via the system
